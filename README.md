Caesar Cipher Brute-force
===

This tool will brute-force Caesar Cipher text.

###Features:
- _brute-force_, statistically: order the most fitting decrypted text that are in English language
- _brute-force_: brute-forces with possible shift key (0-26).
- _enkripsi_: encrypt a text with set shift key.
- _dekripsi_: decrypt a text with set shift key.

###Run:
```sh
python caesar_bruteforce.py
```

also check [CyberChef](https://gchq.github.io/CyberChef/) and [dcode.fr](https://dcode.fr/)
