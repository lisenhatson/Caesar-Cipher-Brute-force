Caesar Cipher Brute-force
===

This tool will brute-force Caesar Cipher text.

### Features:
- brute-force, statistically: order the most fitting brute-forced texts that are in English language.
- brute-force: brute-forces with possible shift key (0-26).
- encrypt: encrypt text with set shift key.
- decrypt: decrypt text with set shift key.

### Run:
```sh
python caesar_bruteforce.py
```

also check [CyberChef](https://gchq.github.io/CyberChef/) and [dcode.fr](https://dcode.fr/).
