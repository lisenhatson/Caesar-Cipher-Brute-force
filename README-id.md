Caesar Cipher Brute-force
===

Alat _brute-force_ ini akan mendekripsi pesan yang dienkripsi dengan _Caesar Cipher_.

Fitur:
- _brute-force_ secara statistik: mencari kata yang dapat dibaca dalam bahasa Inggris.
- _brute-force_ secara biasa: melakukan _brute-force_ dengan menggunakan semua _shift key_ yang mungkin (0-26).
- _enkripsi_: mengenkripsi teks dengan _shift key_ yang terbilang.
- _dekripsi_: mendekripsi teks dengan _shift key_ yang terbilang.

Cara menjalankan:
```sh
python caesar_bruteforce.py
```
